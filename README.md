# Knowledge Base

A self-contained, offline-ready collection of interactive study guides covering core software engineering topics.

**Live site:** [kevintlee01.github.io/knowledge-base](https://kevintlee01.github.io/knowledge-base/)

## Topics

- **[Artificial Intelligence](https://kevintlee01.github.io/knowledge-base/ai-website/artificial-intelligence.html)** — Machine learning, deep learning, NLP, computer vision, generative AI, LLM pipelines, infrastructure, safety & ethics. Includes 8 interactive taxonomy diagrams, code examples, and math foundations.

- **[Microservices Architecture](https://kevintlee01.github.io/knowledge-base/microservices-website/microservices-architecture.html)** — Service decomposition, API gateway, event-driven communication, data patterns, deployment (Docker/K8s), observability, security, and testing. Production-ready code examples throughout.

- **[Data Structures & Algorithms](https://kevintlee01.github.io/knowledge-base/dsa-website/data-structures-algorithms.html)** — Arrays, linked lists, trees, graphs, sorting, searching, dynamic programming, and advanced patterns. Complexity analysis and Python implementations.

## Features

- Single-file HTML pages — no build step, no framework
- Fully self-contained with zero external dependencies (works offline)
- Interactive diagrams, animated flow charts, and clickable taxonomy maps
- Syntax-highlighted code examples with collapsible sections
- Responsive design with dark theme
- KaTeX for math rendering (bundled locally)

## Structure

```
index.html                          # Landing page
ai-website/artificial-intelligence.html
microservices-website/microservices-architecture.html
dsa-website/data-structures-algorithms.html
libs/katex/                         # Local KaTeX library
```

## Running Locally

Open `index.html` in any browser — no server required.
